# Crime Monitoring Intelligence System

A Flask-based AI-powered system that collects crime-related news, analyzes it using NLP, and generates summarized intelligence reports.

## Features
- Crime news collection
- NLP based analysis
- Automated PDF report generation
- Dashboard for monitoring
- Settings and report management

## Tech Stack
- Python
- Flask
- NLP (Transformers / SpaCy)
- HTML, CSS, JavaScript

## Run the Project

pip install -r requirements.txt
python app.py
